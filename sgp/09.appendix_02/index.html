SGP初级使用教程 附录2：Lǒvの钟点工的SGP使用反馈

Terry 发表于 2019-04-20 | 评论数：

一、目标

我用的是最简单的办法，SKYX里进行构图，拍摄一张10秒左右的fit文件，直接导入SGP解析，确定目标计划。

二、AF

昨晚一直在测试AF，目标是IC1796鱼头，窄带Ha滤镜。RB里测得回差in17out20，SGP里填30。

通过（HFR最大值计算工具 http://www.wilmslowastro.com/software/formulae.htm#CFZ）

计算出R=141.G=111.B=103，取最小值103，步数7，步长12。

此图完美。（步数7，步长12，sample size 15，weak）

然后SR告诉我，步数7，步长5.

此图依然完美（步数7，步长5，sample size 15，weak）

但是给出了一个提示，大意是这个HFR2.79可能不是最佳结果，但实际是的。

然后和SR开始纠结于这两个选项。

左边这个是星云强度，右边这个是SGP选取的HFR采样点。

因为是窄带Ha，单张5秒，鱼头都能看出来个大概样子，所以我把左边星云强度这个调到最右边strong。右边这个同样，因为星云强度大，如果选到中间或者左边15。另外有一个经验，手动take one一张，检测HFR，显示检测到的星点数不要多于20个。太多的星点会影响到AF的准确。同时也不能太少，少于5-10个点，同样采样不准确，随机性大，影响AF准确。

HFR曲线变为直线（步数7，步长14，sample size 100，weak）。

最后请SR大神出手，SR大神用我的老七L滤镜

曲线完美。

结论：

窄带nebula 放到weak，sample size 放到15。

LRGBnebula放到weak和strong之间，sample size 放到100.

三、拍摄

我最终选择了（步数7，步长7，sample size 15，weak）模式
