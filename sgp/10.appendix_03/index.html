SGP初级使用教程 附录3：新手设置过程实例

Terry 发表于 2019-04-20 | 评论数：

SGP的设置很多，让新手发怵，以下根据为海盗船长的设置过程，集大家各种经验总结，面向新手做设置介绍。望指正，如果你有心得也请告诉我们。

一、理解SGP的设置

SGP上手设置应先从Tools菜单的Profile开始。

Profile是通用模板，可对不同设备组合设置不同配置方案。每个新的拍摄序列都是基于Profile生成的，序列生成后的控制是在Control Panel里。每个拍摄序列可以设置不同的控制。

三、设置滤镜

这里是自动聚焦的曝光时长。

上图设置时间短了，导致Ha的自动调焦效果不好。一般窄带的建议10～15秒，L通道建议3～5秒。

四、设置自动调焦

选中这里：

再打开自动调焦设置：

上图中选中换滤镜时自动对焦，对于追求完美的同学，即使是A家的滤镜，也最好重新对焦下。另外，建议如果发现有用噪点作为星点的情况，可以通过设置暗场文件去除。

自动对焦时的步长（Step Size）可以借助MDL。先在MDL里设置好每步的移动移动距离（具体可以看yzhzhang的教程），然后让MDL自动聚焦，看MDL的相邻采样点的步长，可以以此作为SGP的Step Size。

设置为9个采样点，再根据MDL里自动计算的，每次移动步长为6。

如果曲线稳定，可以将采样点减少到7个。

还有需要设置回差。

个人经验，回差设置Out方向曲线更漂亮。海盗船长的Out回差是8，实际设置10（根据SR的经验，设置值要比实际值大20～30%）

后来经过实测，发现11最合适。

另外，实践中发现，支架的刚性对自动对焦的成功至关重要！

五、设置赤道仪

拍完自动Park以及自动中天翻转。

自动中天翻转界面上，可以选择等过中天后再拍（即如果下一张照片拍完的时间超过了中天时间就等着过了中天翻转后再拍），过中天翻转后自动对准目标是必须的。

六、设置解析

选择PS2。下面的滤镜选择可以去掉，就用当前滤镜以及滤镜里设置的曝光时长。

七、设置导星

其中最重要的是自动聚焦时要停止导星（红色选择），否则调焦引起的晃动会让导星移动望远镜，使得聚焦一塌糊涂。

（黄色选择）对主CCD和导星OAG一体式的相机有用，PHD通过SGP API控制导星CCD，防止连接超时，看SGP论坛里是SBIG的8300和11K会存在这个问题。对非一体式的相机不需要选择，否则会提示错误。

关于导星开始条件的设置：

第一个是波动多少像素以内认为是平稳，第二个是平稳多久开始拍摄。

其余选项根据自己需要设置。

八、设置失败重试等

开始了序列，如果遇到有云暂时导星失败了SGP就自动停止了，那怎么办？可以在Tools的Options/Sequence Settings里设置：

其中的“attempt to automatically recover the sequence”可以设置如果失败后隔几分钟尝试一次，总共持续多少分钟。

在Genetal Options属性页里还有几个可以设置的地方：

Profile directory是保存Profile配置文件的地方，可以把配置文件拷贝出来备份，具体备份和迁移的方法请参见附录4。

Default image directory是默认的保存照片的目录。

Custon file name pattern是修改默认的文件名组合方式，如果不满意默认的，可以看帮助文件在这里修改。

九、补充：

  1. 上面这些设置搞定后，建议先手动测试下解析和自动对焦，确保单独运行是好的。手动操作界面都在View菜单的Docking Modules里，解析需要用Frame and Focus的Take One拍照后按照第四夜操作，测试自动对焦是Focus Control的Run。
  2. 实际拍摄时，建议先用武总的深空拍摄导航软件策划好目标，知道这个目标啥时候过中天、能拍多久。有了SGP，可以多个目标一直拍到天亮。
  3. 每个序列文件可以独立保存，以后拍摄载入并继续拍摄。继续拍摄时可以双击任务Progess中的数量来设置每项任务已经拍摄了多少张。
  4. 另外，建议闲着没事儿用马赛克工具搜索要拍的目标，SGP会下载星图照片在本地缓存，以后使用就快很多。我把武总全年目标里能拍的都下载了～
