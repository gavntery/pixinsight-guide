SGP初级使用教程 第三夜：自动对焦(AutoFocus)设定及实操

Terry 发表于 2019-04-20 | 评论数：

首先要说明的是，SGP的AutoFocus(自动对焦简写做：AF，下同)机理稍有不同于其他软件的模式，其他软件大都是采用亮星HFR测定对焦的模式，结合V曲线测绘算得。而SGP则是以视场内众多星点目标的HFR“综合值”进行计算。所以，SGP在做AF的时候不需要寻找特定的亮星目标。直接在拍摄视场进行对焦。首先，针对AF的control模块来讲解设定工作。

  • Focus information：调焦器状态信息
  • Focus Control：手动控制对焦菜单
  • Focus Options：设置选项

依次勾选后的设定：

  1. Reverse focuser direction：翻转调焦器进出方向
  2. Focus reminder：调焦动作提醒
  3. Use auto focus：勾选则使用自动对焦，点击Set进入具体设定。这里很重要，具体功能见下面
  4. Auto adjust focus per filter：每次换滤镜时自动调焦
  5. Temperature compensation：温度补偿，如果你不是很确定，建议还是别用吧
  6. Focus Pointing Settings：焦点位置设定
    • Focus：对当前AF滤镜设定合焦位置
    • Set Focus：对所有滤镜采用当前合焦位置设定
  7. 图中最上面Connected to中的Other按钮，点击则进入调焦器设定，可以设置回差等，具体功能见后

自动对焦及其设定（功能编号沿用前面的3）：

3.1 Auto Focus Frequency(自动调焦频率设定)：

  • Auto focus every N frames：每N帧AF
  • Auto focus every N degree change：每变化N度AF（需要电调上有温度传感器）
  • Auto focus every N minutes：每N分钟AF
  • Auto focus on filter change：变换滤镜AF……
  • Auto focus before sequence start(not on resume)：计划开拍AF（而非恢复拍摄时）
  • Auto focus on resume：恢复时AF
  • Auto focus after automatic centering action：自动对中后AF……

3.2 Star Detection：星点侦测

  • Nebula Rejection：对视场中星云目标排除，越强排除的越彻底，一般以视场中星点绿框消失为准
  • Sample Size：AF取样星点多少，这个要结合Image Statistics发现的星点数量来设定，数量如果在100以下，可以适度把数字取小

3.3 Options：选项

  • Exposure time For or OSC ：AF时各滤镜曝光时间设定（点击For子菜单分别对各通道单独设定曝光时间和焦点开始位置）OSC后的曝光时间是针对彩色相机的AF设定
  • Bin：bin模式设定，如果bin1拍摄就选bin1，同样类推
  • Auto Focus Data Points：AF设定步数
  • Step Size：对焦步长；此两项最为关键，步数×步长一定要小于设备的HFR最大值，否则，将无法完成AF，或者严重失准（具体HFR最大值计算参考此网站http://www.wilmslowastro.com/software/formulae.htm#CFZ）
  • Auto focus dialog auto-close…：当对焦完成后，对焦窗口多长时间后自动关闭
  • Disable smart focus：如果物镜有遮挡需要勾选此选项（如牛反，RC等）
  • Auto focus with filter：设定使用哪片滤镜用来自动对焦（通常首选L，其次R，如果各通道都做，需要做最佳曝光测定，此时选择none就行）
  • Crop auto focus frames by：把画面剪切百分之多少来去掉画面周边畸变（主要针对大视场CCD）
  • Apply dark subtraction. Dark library path：应用暗场来优化AF，此处可以指定暗场文件的路径
  • Save Auto Focus packages to: 自动对焦包的保存路径

调焦器设定（功能编号沿用前面的7）：

  • Set absolute focus position for first filter：对第一个滤镜设定为绝对焦点位置（目前一直没有采用这个设置，一般都是开拍前往外走合适的步长——可以参考前夜走得步长来调整，这样的话可能更快）
  • Override maximum steps：手动设置最大调焦步数，做限位用，防止调焦座已经到最外端了还在继续走损坏电调；
  • Focuser backlash compensation：调焦器回差补偿
    • Compensation direction：补偿方向设定（最好是IN，你懂得）
    • Compensation step size：回差补偿值（这个SGP建议一定要比测得实际值大为好，建议大30%吧，如果你的测定准确的话）
  • Use Temper device for temperature input：对调焦器采用温度补偿（不建议使用）

附图：完美V曲线，看到这条曲线，那就恭喜你，成功了！继续下一夜吧～
